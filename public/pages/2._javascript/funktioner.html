Hvad er funktioner i JavaScript?

Funktioner er et grundlæggende koncept i JavaScript og er vigtige af flere grunde:

  • Genbrug af kode: Funktioner giver dig mulighed for at definere en bestemt opgave eller handling, som kan udføres flere gange i din kode. Dette fremmer genbrug og vedligeholdelse af kode.
  • Abstraktion: Ved at opdele komplekse opgaver i mindre funktioner kan du abstrahere detaljerne og gøre din kode mere læselig og forståelig.
  • Modularitet: Funktioner muliggør opdeling af din kode i mindre, håndterbare moduler, hvilket gør det nemmere at organisere og samarbejde i teams.
  • Fejlfinding: Funktioner gør det lettere at isolere og identificere fejl i din kode, da hver funktion udfører en specifik opgave.

Lad os nu se på nogle af de centrale aspekter af funktioner i JavaScript:

Funktionsdeklaration vs. funktionsudtryk

Funktionsdeklaration:

En funktionsdeklaration bruger nøgleordet function til at definere en funktion og tildele den et navn. Funktionen kan derefter kaldes, når som helst i koden, da den er blevet "hoistet" til toppen af sit scope.

Hoisting kan forstås som at det ”bobbler op” , det vil sige at variablen og funktionen ”løftes” til toppen af deres omgivende scope. Derfor vil man kunne deklarere en funktion nedenunder en console.log , fordi de havner i samme callstack. Koden er dog ikke så læsbar, særlig hvis funktionen indeholder en større mængde logik. Da vil det være god kodepraksis og placere funktionen over det sted, hvor den kaldes. 


            console.log(sum(2,2));

            function sum(a, b) {
              return a + b;
            }

Funktionsudtryk:

En funktionsudtryk opretter en anonym funktion og tildeler den til en variabel. Funktionen kan kun kaldes, efter variablen er blevet tildelt. 

const sum = function(a, b) {
        return a + b;
      };

Forskellen ligger i, hvordan de håndteres under hoisting og tilgængelighed i forskellige dele af koden.

Argumenter og parametre

Parametre er variabler, der fungerer som input til en funktion. 


    function sum(a, b) {

    return a + b;
    }

Argumenter er de faktiske værdier, der sendes til funktionen, når den kaldes. 


    const resultat = sum(2, 3);

Returneringsværdier

Funktioner i JavaScript kan returnere værdier ved hjælp af return-nøgleordet. 


    function sum(a, b) {
      return a + b;
    }

    const resultat = sum(2, 3); 
    // resultat vil være 5

Anonyme funktioner

Anonyme funktioner er funktioner uden navn og bruges ofte som argumenter til andre funktioner som callback-funktioner. 


    setTimeout(function() {
      console.log('Dette er en anonym funktion');
    }, 1000);
