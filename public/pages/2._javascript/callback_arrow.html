Arrow funktioner

Arrow-funktioner er en kompakt og elegant måde at definere funktioner på i JavaScript. De blev introduceret i ECMAScript 6 (ES6) og er især nyttige til kortere funktioner og callback-funktioner. Arrow-funktioner adskiller sig fra traditionelle funktioner ved at bruge en kortere syntaks og ved at binde deres eget 'this' til det omsluttende scope.

Her er et eksempel på en arrow-funktion: 

     
      const addArrow = (a, b) => a + b;
      
      console.log(addArrow(3, 5)); // Resultat: 8

Bemærk den korte syntaks og fraværet af nøgleordet 'function'. Arrow-funktioner returnerer automatisk resultatet, når der ikke er brugt krøllede parenteser.

En anden vigtig egenskab ved arrow-funktioner er, at de bevarer det omsluttende scopes 'this'. Dette gør dem velegnede til brug som callback-funktioner og i kontekster, hvor du ønsker at undgå 'this'-problemer. 


      const person = {
        name: "Alice",
        sayHello: function() {
          setTimeout(function() {
            console.log("Hej, jeg er " + this.name); 
            // Vil give fejl, 'this' er ikke person-objektet
          }, 1000);
        },
        sayHi: function() {
          setTimeout(() => {
            console.log("Hej, jeg er " + this.name); 
            // Arrow-funktion bevarer 'this' fra det omsluttende scope
          }, 1000);
        }
      };
      
      person.sayHello(); // Vil give fejl
      person.sayHi(); // Vil logge "Hej, jeg er Alice" efter 1 sekund

Arrow-funktioner er en nyttig tilføjelse til JavaScript og kan gøre din kode mere læselig og vedligeholdelig, især når du arbejder med kortere funktioner og callbacks.

Callback-funktioner

Callback-funktioner er en kraftfuld del af JavaScript, der giver dig mulighed for at definere en funktion separat og derefter overføre den som en reference til en anden funktion som et argument. Dette gør det muligt at genbruge og opdele din kode i mindre, mere forståelige stykker.

Callback-funktioner bruges ofte i asynkron programmering, hvor operationer ikke venter på resultater med det samme, men i stedet udføres i baggrunden. Når resultatet er klar, kaldes callback-funktionen.

Her er et grundlæggende eksempel på brugen af callback-funktioner: 


      function genericActionPerformer(genericAction, name) {
        return genericAction(name);
      }
      
      // Arrow-funktion!
      const jump = (name) => `${name} is jumping`;

      // "Lasse is jumping"
      // jump bruges som en callback-funktion!
      console.log(genericActionPerformer(jump, "Lasse"));

Du kan også lave det samme på én linje: 


      console.log(genericActionPerformer((name) => `${name} is sleeping`, "Daniel"));

Callback-funktioner giver stor fleksibilitet og kan bruges i forskellige sammenhænge, såsom event-håndtering og asynkron kommunikation med servere.
