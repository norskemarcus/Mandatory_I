Clean code

Hvorfor er clean code vigtigt?

  • Læsbarhed: Clean code er let at læse, forstå og vedligeholde. Dette er vigtigt, da kode typisk læses af mennesker (inklusive dig selv i fremtiden), der skal forstå, hvad koden gør.
  • Fejlfinding: Clean code gør det lettere at spotte og rette fejl. Når koden er ryddelig og organiseret, bliver fejl mere synlige og nemmere at løse.
  • Samarbejde: Når flere udviklere arbejder sammen på et projekt, gør clean code samarbejdet lettere. Andre kan nemt forstå og bygge videre på din kode.
  • Vedligeholdelse: Projekter ændrer sig over tid. Clean code er mere fleksibel og lettere at tilpasse nye krav.
  • Effektivitet: Skrive clean code tvinger dig til at tænke mere omhyggeligt over, hvordan du løser problemer, hvilket kan føre til mere effektive løsninger.

Principper for clean code

  • Navngivning: Brug beskrivende og meningsfulde navne til variabler, funktioner og klasser. Undgå forkortelser, medmindre de er velkendte i branchen.
  • Kortfattet kode: Hold funktioner og metoder korte og fokuserede. Undgå lange funktioner, der gør for meget. Del komplekse opgaver op i mindre funktioner.
  • Kommentarer: Skriv klare og forståelige kommentarer, der forklarer komplekse eller uventede kodestykker. Undgå unødvendige kommentarer, da koden selv bør være selvforklarende.
  • En funktion, én opgave: Funktioner og metoder bør kun udføre én specifik opgave. Dette gør dem lettere at teste, forstå og vedligeholde.
  • Formatering og indrykning: Hold koden formateret konsistent. Brug indrykning for at fremhæve kodeblokke og gør koden let at scanne visuelt.
  • Undgå unødvendig kompleksitet: Hold det simpelt. Undgå dybt nedarvede strukturer og komplekse løsninger, medmindre det er absolut nødvendigt.
  • Testning: Skriv tests for din kode for at sikre, at den fungerer som forventet. Testdrevet udvikling (TDD) kan hjælpe med at opnå clean code.
  • Refaktorering: Løbende forbedr kodebasen ved at fjerne duplikation og forbedre strukturen. Refaktorér, når du ser mulighed for at gøre koden renere.
