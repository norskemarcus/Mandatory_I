Entry point og package.json

Et entry point fortæller Node.js hvilken JavaScript fil som skal køres, når applikationen starter. Dette entry point omtales gerne som main i package.json filen og i eksempelet under kan man se at filen er 'app.js': 


  {
    "name": "mandatory1",
    "version": "0.0.1",
    "description": "Mandatory I for Marcus Holje",
    "type": "module",
    "main": "app.js", // Entry point
  }
